# jc1c-barcode
## Описание
Данная компонента сделана на основе [jc1c](https://github.com/kaliuzhnyi/jc1c).

Предназначена для генерирования и для чтения штрихкодов.
Допустимые типы штрихкодов:
- AZTEC
- CODABAR
- CODE 39
- CODE 93
- CODE 128
- DATA MATRIX
- EAN 8
- EAN 13
- ITF
- PDF 417
- QR
- UPC A
- UPC E
